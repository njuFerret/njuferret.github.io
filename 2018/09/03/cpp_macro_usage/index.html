3) #和##的一些应用特例

(1) 合并匿名变量名

例：ANONYMOUS(static int); 即: static int _anonymous70;，70表示该行行号;

宏的层次 转换前 方向 转换后
第1层： ANONYMOUS(static int); –> __ANONYMOUS0(static int, __LINE__);
第2层： __ANONYMOUS0(static int, __LINE__); –> ___ANONYMOUS1(static int, _anonymous, 70);
第3层： ___ANONYMOUS1(static int, _anonymous, 70); –> static int _anonymous70;

即每次只能解开当前层的宏，所以__LINE__在第二层才能被解开;
